# SajiloSchool - Passwords to Remember

The system uses Django Auth, so all passwords are securely hashed (e.g., pbkdf2_sha256). We cannot extract plaintext passwords from the database. However, here are the details for your test/seeded users:

1. **Superadmin / Admin Users:**
Since these passwords were created during setup, we cannot retrieve them. If you cannot remember the admin password, you can reset it by running:
`python manage.py createsuperuser`
or by opening the Django shell and setting a new one:
`python manage.py shell -c "from django.contrib.auth import get_user_model; User=get_user_model(); u=User.objects.get(username='superadmin'); u.set_password('newpassword'); u.save()"`

2. **Student Users (from the seed script):**
All 700 seeded students were created with the default password:
Password: `password123`
Username format: `std_1_0_1234` (varies by class and random number)

3. **Tenant/Client Databases:**
Every tenant has independent users. If you need to access a specific school, make sure you know the admin credentials for that specific tenant.
